## Parameters

### Required Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `prompt` | string | Detailed description of the desired output |

### Optional Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `aspect_ratio` | string | "1:1" | Output aspect ratio |
| `style` | string | "auto" | Generation style preference |
| `quality` | string | "standard" | Quality level: "standard", "high" |
| `seed` | number | random | Random seed for reproducible results |

## Examples

### Creative Content Generation

```javascript
const response = await fetch('/api/v1/minimax/hailuo-02', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer your_api_key',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    prompt: 'A surreal landscape where trees grow musical instruments instead of leaves',
    aspect_ratio: '16:9',
    style: 'artistic',
    quality: 'high'
  })
});

const result = await response.json();
console.log('Generated image:', result.images[0].url);
```

### Python Example

```python
import requests

def generate_with_minimax(prompt, **kwargs):
    response = requests.post(
        'https://firemoon.studio/api/v1/minimax/hailuo-02',
        headers={
            'Authorization': 'Bearer your_api_key',
            'Content-Type': 'application/json'
        },
        json={
            'prompt': prompt,
            'aspect_ratio': kwargs.get('aspect_ratio', '1:1'),
            'style': kwargs.get('style', 'auto'),
            'quality': kwargs.get('quality', 'standard'),
            **kwargs
        }
    )

    response.raise_for_status()
    return response.json()

# Usage
result = generate_with_minimax(
    'An abstract representation of artificial intelligence as a cosmic entity',
    aspect_ratio='4:3',
    quality='high'
)

print(f"Generated image: {result['images'][0]['url']}")
```
